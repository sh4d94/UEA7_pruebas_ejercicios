## Objetivo del repositorio. <br>

El presente repositorio tiene como objetivo central **organizar, almacenar y facilitar el acceso** a los ejercicios, experimentaciones y entregas correspondientes al apartado tecnológico, impartido por el Dr. Hernani Villaseñor, de la **UEA VII Producción y Comunicación en la Cultura Colaborativa**. <br>

Aquí se recopilan tanto las actividades realizadas durante el curso como las diversas pruebas vinculadas al eje integrador de la asignatura obligatoria, con la intención de documentar de forma estructurada el proceso de aprendizaje y creación técnica. Además de cumplir con una función archivística, este repositorio busca convertirse en un **espacio dinámico de trabajo**, donde los avances, errores, prototipos y soluciones queden registrados como parte de mi proceso formativo. <br>

De esta forma, la intención es mostrar cómo las herramientas digitales, (desde un pequeño script hasta desarrollos más complejos) se integran progresivamente en la construcción de un proyecto trabajado en equipo dentro del ámbito del arte colaborativo y corpóreo. En este sentido, el repositorio no solo opera como un contenedor estático de evidencias, sino también como un **laboratorio de documentación**, donde cada ejercicio representa un paso dentro del desarrollo técnico-creativo del trimestre. Su función final es permitir la revisión, continuidad y recuperación de materiales útiles para futuros proyectos, ejercicios y tareas.


## Organización del repositorio (200 palabras).<br>

La estructura del repositorio está diseñada para garantizar una navegación clara y una clasificación ordenada de los materiales. Se divide en tres bloques principales: ejercicios, pruebas y documentación. Cada carpeta responde a una lógica funcional que permite identificar rápidamente el tipo de contenido que contiene y su relación con el proceso formativo.

En la carpeta /ejercicios se agrupan las actividades realizadas dentro del curso, organizadas por semanas o temas. Cada ejercicio cuenta con su propio espacio, donde se incluyen scripts, capturas, notas y cualquier archivo complementario. Esta clasificación facilita rastrear la evolución técnica a lo largo del semestre.

La carpeta /pruebas funciona como un espacio de experimentación, donde se registran prototipos, tests y versiones preliminares relacionados tanto con las tareas del curso como con el proyecto integrador. Aquí se concentran aquellas ejecuciones que permiten explorar ideas, verificar funcionalidades y consolidar aprendizajes técnicos.

Finalmente, la carpeta /documentación contiene notas, reflexiones, bitácoras y material conceptual que complementa los procesos técnicos. Su objetivo es mantener visible la dimensión analítica y metodológica del trabajo. En conjunto, esta organización busca mantener el repositorio como un recurso claro, accesible y útil para revisiones posteriores.


## Experiencia personal de crear y gestionar tu repositorio (200 palabras).<br>

La experiencia de crear y gestionar este repositorio ha sido un ejercicio significativo en términos de organización, disciplina técnica y comprensión del flujo de trabajo digital. Al inicio, el uso de GitHub representó un reto, especialmente en la estructuración de carpetas y en la toma de decisiones sobre cómo documentar cada paso del proceso. Sin embargo, conforme avanzaron las actividades, el repositorio se convirtió en una herramienta fundamental para visualizar mi propio progreso y mantener un control más claro de las versiones y modificaciones realizadas.

Gestionar este espacio me permitió reconocer la importancia de documentar no solo los resultados, sino también las pruebas fallidas, los errores recurrentes y las soluciones encontradas en el camino. Este enfoque hizo del repositorio un espacio honesto y funcional, donde la experimentación y el registro riguroso se vuelven parte natural del trabajo creativo.

Además, la práctica constante con GitHub favoreció una mayor conciencia sobre las dinámicas de colaboración digital, el mantenimiento de código y la preservación de proyectos a largo plazo. En síntesis, esta experiencia consolidó habilidades técnicas esenciales y fortaleció mi capacidad para organizar procesos complejos dentro del ámbito académico y artístico.
